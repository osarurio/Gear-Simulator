## 🎮 遊び方（UIの操作方法）

1. **配置する**
   画面横の「ライブラリ」から好きな歯車を選び、メインのエディター画面へ**ドラッグ＆ドロップ**で配置します。
2. **噛み合わせる**
   エディター上の歯車同士を近づけると、自動的に**スナップ（吸着）**して綺麗に噛み合います。
3. **動かす**
   画面内の **「Run」ボタン** を押すと、モーターギアを起点にシミュレーションがスタートします。

---

## 🧰 使用できる歯車の種類

| ギア名 | 特徴と機能 |
| :--- | :--- |
| **⚡ モーターギア** | すべての動力の源となるギアです。メニューから**回転の速さを自由に変更**することができます。 |
| **⚙️ スンダードギア** | 最も標準的な普通のギアです。動力の伝達や回転方向の反転に使います。 |
| **🔄 2段ギア** | 大小2つの歯車が同軸上でくっついた特殊なギアです。これを使うことで、**回転数（スピード）とトルク（回転力）の調整（変速）**が可能になります。 |
| **🔒 ラチェットギア** | **指定した一方の方向だけ**に回転するギアです。逆方向から力がかかった場合は回転をロック（無視）します。 |
